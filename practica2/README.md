# Pràctica 2 – Monitorització Ambiental d’un CPD

## Objectiu
Crear un sistema amb ESP32 que controli la temperatura i la llum d’una sala de servidors, activant alertes automàtiques i permetent consultar dades via Serial.

## Contingut del repositori
- /src → Codi font (main.ino)
- /doc → Documentació, imatges, PDF i vídeo
- README.md → Explicació general
