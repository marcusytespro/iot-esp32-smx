# Projecte IoT – Monitorització de CO₂ a l’Aula

## Objectiu

Crear un sistema amb ESP32 que monitoritzi els nivells de CO₂ i la qualitat de l’aire dins d’una aula, mostrant les dades en temps real mitjançant una pàgina web.

## Contingut del repositori

* /src → Codi font (main.ino)
* /doc → Documentació, imatges, PDF i vídeo
* README.md → Explicació general
